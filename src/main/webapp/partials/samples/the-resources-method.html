The _resources method

This sample explains how to use the _resources method in its simplest way. For more information about the _resources method see this documentation. The following list shows the parent category of the second category.

This sample just shows one way of using the resources method. The _resources method has more features like adding URL parameters to the requests, or return the available resources. And you are also able to use all the features of the returned Angular $resource object.

  • {{parentCategory.name}}
    {{parentCategory._links.self.href}}
SpringDataRestAdapter.process($http.get('/rest/categories')).then(function (processedResponse) { var parentCategoryResource = processedResponse._embeddedItems[1]._resources("parent"); $scope.parentCategory = parentCategoryResource.get(); });
