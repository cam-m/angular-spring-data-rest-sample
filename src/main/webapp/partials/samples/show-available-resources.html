Show available resources

This sample explains how to get an object with all available resources on the root of the back end REST endpoint or on any other available resource.

  • Available resources: {{availableResources.parameters}}
SpringDataRestAdapter.process($http.get('/rest')).then(function (processedResponse) { $scope.processedResponse = angular.toJson(processedResponse, true); $scope.availableResources = processedResponse._resources(); });
